无重复字符最长子串

发表于 2022-03-28 更新于 2022-03-29 分类于 算法

题目描述

无重复字符最长子串

一刷

  1. 使用HashMap保存子串数据及其下标, 子串起始下标start初始值为0
  2. 逐步增加子串长度, 遇到重复字符, 获取已保存在map中的重复字符的下标index, 令start = index + 1, 同时记录当前子串最大长度
  3. 重复步骤2, 直到遍历完整个字符串

二刷

一刷中的while循环有些多余了, 去除while循环, 思路与一刷大致类似

三刷

前面的思路中, 遇到重复字符, 就将map中保存的字符全部清空, 然后从index + 1开始进行下一次子串循环, 实际上, 只需要将map中下标index之前的值删除(注意,重复值不删除), 然后从出现重复字符的下标位置 + 1开始进行下一轮子串循环就可以了

官方解法

官方解法提到了滑动窗口的概念, 与我的解法吻合, 这个滑动窗口的思想有点熟悉, 待溯源…

  • 与我的[一刷][#一刷]想法大体一致, 与二刷或三刷相比, 不同的其实就是官解中左指针一步一步移动的, 最终目的就是移动到重复字符的位置, 将重复字符删去
  • 两数取较大值, 使用Math.max(a, b)方法更为简洁明了,易于阅读
  • 由于String类底层就是使用char[]保存字符串数据, 使用String类的charAt(index)方法可直接取得字符, 那么三刷中使用split方法将字符串分割为字符串数组有些多余了, 故做了一次优化
